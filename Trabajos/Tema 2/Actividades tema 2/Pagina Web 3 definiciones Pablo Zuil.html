Tipos de lenguaje marcado

Definiciones HTML, XML Y JSON

HTML

- HMTL Es el lenguaje de marcado que sirve para escribir y estructurar las páginas web. Coloca etiquetas alrededor de texto, imágenes, enlaces, formularios y otros elementos multimedia para organizar el contenido. HTML no es un lenguaje de programación, sino una de marcado, lo que quiere decir que no indica cómo se encuentra un contenido, sino cómo se estructura para dársela educación del mismo.

XML

- XML es un formato usado para organizar y almacenar información de manera que tanto personas como ordenadores puedan entenderla. Funciona como una serie de etiquetas que rodean los datos, un poco como cuando pones títulos en un cuaderno para saber qué contiene cada sección. Por ejemplo, si estás guardando información sobre un libro, podrías tener una etiqueta que dice "título" y dentro de esa etiqueta, el nombre del libro. A diferencia de otros lenguajes como HTML, en XML puedes inventar tus propias etiquetas para describir lo que sea necesario. Se usa mucho para enviar datos entre sistemas o aplicaciones.

JSON

- JSON es una forma sencilla de organizar datos para que los sistemas o aplicaciones puedan compartir información fácilmente. Funciona como una lista de pares donde cada cosa tiene un nombre y un valor, por ejemplo, como una lista de compras: "manzanas" (nombre) y "5" (valor). Aunque se basa en la forma en que se escribe en JavaScript, se puede usar con muchos lenguajes de programación. Es muy común en páginas web y aplicaciones porque es fácil de leer y entender tanto para las personas como para los ordenadores.
